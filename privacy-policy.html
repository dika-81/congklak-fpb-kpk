Kebijakan Privasi

Congklak FPB KPK
Nama paket: id.upi.pgsd.congklakfpbkpk
Berlaku sejak: 13 Juli 2026

Congklak FPB KPK adalah aplikasi permainan edukatif untuk membantu memvisualisasikan konsep FPB, KPK, dan algoritma Euclid melalui algoritma selisih lumbung.

1. Pengembang

Aplikasi Congklak FPB KPK dikembangkan dan dikelola oleh Andika Arisetyawan. Kebijakan Privasi ini menjelaskan cara aplikasi menangani informasi pengguna.

2. Data yang Dikumpulkan

Aplikasi ini tidak meminta pengguna membuat akun dan tidak secara sengaja mengumpulkan, menyimpan, menjual, atau membagikan data pribadi maupun data sensitif pengguna.

Aplikasi tidak meminta nama, alamat, nomor telepon, lokasi, kontak, foto, rekaman, informasi keuangan, atau identitas pribadi lainnya.

3. Penyimpanan Lokal dan Cache

Aplikasi dapat menggunakan cache atau penyimpanan lokal pada perangkat untuk memuat konten dan mendukung penggunaan aplikasi. Informasi tersebut tetap berada pada perangkat pengguna dan dapat dihapus dengan membersihkan data aplikasi atau mencopot pemasangan aplikasi.

4. Izin Perangkat

Aplikasi tidak memerlukan akses ke lokasi, kamera, mikrofon, kontak, pesan, telepon, foto, atau berkas pribadi pengguna untuk menjalankan fungsi utamanya.

5. Iklan dan Transaksi

Aplikasi tidak menampilkan iklan, tidak menyediakan pembelian dalam aplikasi, dan tidak memproses transaksi keuangan.

6. Layanan dan Infrastruktur Pihak Ketiga

Konten aplikasi disediakan melalui layanan hosting GitHub Pages. Saat perangkat terhubung ke layanan tersebut, penyedia infrastruktur dapat memproses informasi teknis standar, seperti alamat IP, jenis perangkat, atau catatan akses, sesuai kebijakan privasi mereka. Pengembang tidak menggunakan informasi tersebut untuk membuat profil pengguna, periklanan, atau penjualan data.

7. Privasi Anak

Aplikasi ini dapat digunakan sebagai media pembelajaran oleh siswa, termasuk anak-anak. Aplikasi tidak meminta anak memberikan data pribadi, tidak menyediakan komunikasi antarpengguna, dan tidak menampilkan iklan berbasis minat.

Orang tua, wali, atau guru disarankan mendampingi penggunaan aplikasi sesuai usia dan kebutuhan belajar anak.

8. Keamanan Data

Karena aplikasi tidak menyediakan formulir pengumpulan data dan tidak meminta akun pengguna, pengembang tidak menyimpan basis data pribadi pengguna. Pengembang tetap berupaya menjaga konten dan kode aplikasi agar aman serta sesuai dengan tujuan pendidikan.

9. Penyimpanan dan Penghapusan Data

Pengembang tidak menyimpan data pribadi pengguna pada server milik pengembang. Data cache atau data lokal pada perangkat dapat dihapus melalui pengaturan aplikasi atau dengan mencopot pemasangan aplikasi.

10. Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui apabila terdapat perubahan fungsi, teknologi, atau ketentuan yang berlaku. Versi terbaru akan ditampilkan pada halaman ini beserta tanggal berlakunya.
